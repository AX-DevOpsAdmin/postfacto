# postfacto

## Instructions

1. Install Ruby 2.4.0
2. Install bundler
3. Install rspec
4. Install and start Postgres
5. Instal Node 6
6. Install npm 3.10.10 (`npm install -g npm@3.10.10`)
7. Install gulp
8. Install chromedriver
